CSS 2 JUCE

Bounds

Generates a .h file filled with juce::Rectangle<float>'s based on your selected layers. Your layer heirarchy is preserved through the use of namepsaces.

Instructions:

  1. Select 1 or more layers within your project
  2. Click 'run'
  3. Modify it below and hit save!

Component.h

Generates a juce::Component .h header file based on the currently selected layer. Direct children are added as private members.

Instructions:

  1. Select a single layer within your project
  2. Click 'run'
  3. Modify it below and hit save!

Component.cpp

Generates a juce::Component .cpp declaration file based on the currently selected layer. Direct children of the selected layer are made visible and have their size automatically set by calling addAndMakeVisible(m_some_child) within the constructor and m_some_child.setBounds(GUI::Dimensions::Bounds) within the resize() method.

Instructions:

  1. Select a single layer within your project
  2. Click 'run'
  3. Modify it below and hit save!
